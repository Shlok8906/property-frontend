## üìû Quick Reference

### Create a Lead
1. Go to Admin ‚Üí Leads
2. Click "Add Lead"
3. Fill form with required fields
4. Click "Save Lead"

### Manage Lead Status
1. Find lead in table
2. Click "Actions"
3. Select status
4. Status updates immediately

### Add Notes to Lead
1. Click notes icon (‚≠ê)
2. View history
3. Add new note
4. Click "Save Note"

### Filter Leads

2. Use priority filter (Hot, Warm, Cold)
3. Use status filter (Interested, Followup, Rejected)
